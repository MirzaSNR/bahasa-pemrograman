## Variabel dan Tipe Data di Kotlin

Variabel                
-> berguna untuk menyimpan nilai

Variabel di Kotlin memiliki 2 tipe :
- Mutable   (val)       -> Dapat dirubah nilainya
- Imutable  (var)       -> Tidak Dapat dirubah nilainya

Cara untuk membuat Variabel :
1. Penulisan nama variabel menggunakan gaya = CamelCase
2. Nama Variabel tidak diperbolehkan menggunakan simbol
   *kecuali menggunakan _ (underscore)
3. Nama Variabel tidak diperbolehkan dengan awalan angka dan simbol


Contoh dari membuat Variabel :

var fullName = "Muhammad Rizki"
val age = 24


Tipe Data
-> jenis nilai yang disimpan didalam sebuah variabel

Tipe Data di Kotlin :
- String            ->  Text                -> "Hacktoberfest"
- Int(integer)      ->  Angka Bulat         -> 2020
- Boolean           ->                      -> True dan False
- Char              ->  Karakter            -> 'B'
- Float             ->  Angka Pecahan       -> 22.5

Contoh dari membuat Variabel dengan Tipe Data :

var fullName: String = "Muhammad Rizki"
val age: Int = 24
